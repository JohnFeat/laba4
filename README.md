# laba4

Полнофункциональное десктопное приложение для управления библиотечным каталогом с использованием SQLite базы данных
Управление каталогом книг
Добавление новых книг с различными типами выдачи
Редактирование существующих записей
Удаление книг из каталога
Поиск книг с минимальным штрафом за просрочку

Работа с данными
* SQLite база данных - автоматическое создание и управление
* Экспорт/импорт данных в текстовые файлы
* Автосохранение всех изменений в БД
* Статистика в реальном времени
* Бизнес-логика

Два типа выдачи книг: стандартная и с продлением
* Гибкая система штрафов: скидки за продление срока
* Умный поиск: нахождение книг с минимальными штрафами
